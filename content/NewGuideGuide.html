2. Adding a New Guide

    +-------------------+
    | [+] Add New Guide |
    +-------------------+
          |
          v
    +-------------------+
    | Title: [        ] |
    | Lang:  [Select v] |
    | Content:          |
    | +---------------+ |
    | |               | |
    | |    Markdown   | |
    | |     Editor    | |
    | |               | |
    | +---------------+ |
    | [  Save Guide   ] |
    +-------------------+
  1. Click "Add New Guide" button
  2. Enter the guide title
  3. Select the language (en, he, ar, ru)
  4. Write your guide content in Markdown format
  5. Click "Save Guide" to add it to the system

7. Localizing Guides

    EN -----> [Translate] -----> HE
    |                            |
    v                            v
    AR <---- [Translate] <----- RU

For each guide:

  1. Create versions in all supported languages (en, he, ar, ru)
  2. Use the same file name with language suffix (e.g., lab-safety-en.html, lab-safety-he.html)
  3. Ensure proper RTL formatting for Hebrew and Arabic versions

8. Publishing and Updating

    [Draft] ---> [Review] ---> [Publish]
      ^                           |
      |                           |
      +---------------------------+
             [Update]

Guide lifecycle:

  1. Create or edit guide (Draft stage)
  2. Submit for review
  3. Publish when approved
  4. Update as needed, returning to Draft stage

LEGO HubLab Tip

Think of each guide as a unique LEGO brick. The maintenance page is your sorting tray, helping you organize and connect these bricks into a comprehensive knowledge structure!
